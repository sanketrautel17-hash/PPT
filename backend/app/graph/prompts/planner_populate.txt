## CRITICAL: Placeholder Key Rules
The `placeholders` object in your JSON output MUST use the EXACT `idx` integer values shown in the
"Template Details" section above as string keys — NOT sequential numbers.
For example, if Template Details shows "Placeholder idx=10 (TITLE)" and "Placeholder idx=13 (BODY)",
your JSON must be: `"placeholders": {{"10": "Your Title Here", "13": ["bullet 1", "bullet 2"]}}`.
Using wrong keys (0, 1, 2 when the template has 10, 13) means NO text will be replaced.
